# Conformance vectors (VRP v0.1)

Canonical test proofs for offline verifier CI. Generated by `generate_vectors.py` (P0 reference implementation).

## Files

| Vector | Expected |
|--------|----------|
| `valid.vrp.json` | Verify PASS, verdict PASS |
| `tampered.vrp.json` | Verify FAIL (bad signature) |
| `drop.vrp.json` | Verify FAIL, verdict FAIL, 1 missing |
| `dup.vrp.json` | Verify FAIL, verdict FAIL, duplicate evidence |
| `mutated.vrp.json` | Verify FAIL, verdict FAIL, mutation evidence |

Each `.vrp.json` has a sibling `.expected.json` with machine-readable expectations.

## Regenerate

```bash
python conformance/generate_vectors.py
python conformance/validate_p0.py
```

## Test public key

`test-key.pub.b64` — Ed25519 public key (deterministic P0 test key only).

## Schema

`vrp-0.1.schema.json` — JSON Schema draft 2020-12 for structural validation.
